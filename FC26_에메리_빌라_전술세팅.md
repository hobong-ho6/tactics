# FC26 에메리 아스톤 빌라 전술 세팅 종합 정리

> **기준 시즌:** 2025-26 (프리미어리그 4위, 유로파리그 우승 – 프라이부르크 3-0)
> **검증 방식:** FC26 인게임 롤/포커스 원문(FUT.GG / FUTBIN) + 실제 25-26 시즌 전술 분석·히트맵·경기 실측
> **목적:** 다른 세션에서 재사용 가능한 레퍼런스 문서

## 1. 팀 전술 (Team Tactics)

| 항목 | 기본 설정 | 근거 |
|---|---|---|
| 포메이션 | **4-2-3-1** (수비 시 4-4-2 자동 전환) | 에메리 실제 기본 틀. 수비 시 10번이 ST 옆으로 올라가 4-4-2 형성 |
| 빌드업 스타일 | **Balanced** | 후방 전개 + 상황적 다이렉트 혼용. Counter처럼 무작정 돌진 안 함 |
| 수비 접근법 | **Balanced / 라인 높이 55~60** | 게겐프레싱 아닌 미드블록. 콤팩트 유지 후 최적 순간 탈취 |

### 상황별 조정
| 상황 | 수비 접근법 / 라인 | 빌드업 | 실제 근거 (PPDA 등) |
|---|---|---|---|
| 강팀 상대 (리버풀·시티급) | **Deep / 40~45** | Balanced | 리버풀전 PPDA 20.43, 시티전 14.44로 물러나 흡수 |
| 약팀·로우블록 상대 | **High / 65~70** | Balanced | 울브스전 4.99, 선덜랜드전 5.78로 전방 압박 극대화 |
| 리드 관리 (후반 잠그기) | **Deep** | **Short Passing** | 4-4-2 콤팩트 블록으로 템포 죽이기 |

- **Aggressive 미사용:** 에메리는 에너지를 아끼는 선택적 압박. 게겐프레싱 팀 아님. 약팀 상대 High까지가 상한.

## 2. 주전 XI 롤/포커스 (기본 = 로저스 10번 구성)

**베스트 XI:** 마르티네스 – 캐시·콘사·파우토레스·마첸 – 카마라·틸레만스 – 맥긴·로저스·부엔디아 – 왓킨스

| 포지션 | 선수 | 롤 / 포커스 | 핵심 근거 |
|---|---|---|---|
| GK | 마르티네스 | **Sweeper Keeper / Balanced** | 박스 밖 수비 액션 90분당 2.18회(리그 2위). 스위핑+빌드업 옵션 겸비. Ball-Playing Keeper는 라인에 머물러 스위핑 죽음 → 부적합 |
| RB | 캐시 | **Wingback / Balanced** | "전진 지원 후 복귀" 왕복형. 아스날전 파포스트 크로스 득점. Fullback은 "뒤에 머물러" 오버랩 없음 → 부적합 |
| CB(우) | 콘사 | **Defender / Defend** | 스피드·1대1·리커버리·라인 유지가 정체성. "no-nonsense 정통 수비수". 파우와 의도적 대비 |
| CB(좌) | 파우 토레스 | **Ball-Playing Defender / Build-Up** | 캐리 전진거리 리그 1위(2,344m), 딥 전진패스 팀 최다, 미드필드 스텝업. 빌드업 개시자 |
| LB | 마첸 | **Wingback / Balanced** | 양방향 풀백("수비 회피 안 함"). 창출력 팀 1위지만 규율 유지. Attacking Wingback은 "수비 축소·복귀 안 함" → 부적합 |
| CDM | 카마라 | **Holding / Defend** | "라인 브레이킹 패스 안 함, 단순 유지"=청소부. 있을 때 승률 57.9% vs 없을 때 40.8%. 잔류하여 역습 차단 |
| CDM | 틸레만스 | **Deep-Lying Playmaker / Build-Up** | 후방 빌드업 촉매 + 수직 전진. Roaming(횡이동)은 부적합 |
| RM | 맥긴 | **Inside Forward / Balanced** | 왼발 인버티드 컷인 + 근면. 안으로 좁혀 중앙 과부하 |
| CAM | 로저스 | **Playmaker / Roaming** | 아래 상세 (§4) |
| LM | 부엔디아 | **Inside Forward / Balanced** | 오른발 컷인 창조형. 클러치 퍼포머 |
| ST | 왓킨스 | **Advanced Forward / Versatile** | 좌우 드리프트 + 빌드업 연계. Attack(중앙 고정, Limited Width)은 부적합 |

### 좌우 인버티드 구조 (중요)
- **맥긴 = 오른쪽(왼발잡이), 부엔디아 = 왼쪽(오른발잡이)** — 둘 다 인버티드 윙어로 안쪽 컷인
- 양쪽 동시 내향 → 중앙 과부하 → 왓킨스가 좌우로 흐를 공간 창출
- 측면 폭은 풀백(캐시 오버랩)이 담당

## 3. 미드필드 조합 서열 (오나나 가용성에 따라 결정)

> **핵심 발견:** 에메리 진짜 1순위는 "카마라+틸레만스 피벗"이 아니라 **"카마라+오나나 피벗 + 틸레만스 10번"**. 11연승 구간 실측 최상 조합. 오나나 가용성이 스위치.

### 🥇 1순위 (풀전력, 시즌 최상 실측)
- 카마라: **Holding / Defend** (잔류 청소부)
- 오나나: **Holding / Roaming** (올라가 점프 압박)
- 틸레만스: **CAM Playmaker / Balanced** (선발 10번)
- 로저스: **LM Inside Forward / Balanced** (좌측 이동)
- 맥긴: RM Inside Forward / Balanced
- **근거:** "최고 퍼포먼스는 이 4미들 동시 기용"(맥긴 우측+틸레만스 10번+카마라·오나나 중앙). "에메리 1순위 미드필드의 열쇠는 오나나 가용성"

### 🥈 2순위 (오나나 부재 = EL 결승 최종형 = 현 기본 세팅)
- 카마라: Holding / Defend
- 틸레만스: **DLP / Build-Up** (피벗)
- 로저스: **CAM Playmaker / Roaming** (10번)

### 🥉 3순위 (카마라 부재)
- 오나나 **Holding / Defend** 승격 + 틸레만스 DLP/Build-Up
- 빅매치: **린델뢰프 Centre-Half / Defend** + 틸레만스 (EL 결승 검증)
- ※오나나는 "메인"보다 카마라 옆에서 최고라는 실측 유의

### 오나나 포커스 상세
- **기본 = Roaming:** rangy 이동형 + 양방향 엔진. 카마라 잔류 + 오나나 점프 압박이 에메리의 "한 명이 상대 6번을 점프 압박" 구조 재현. 듀얼 61.6%·태클+인터셉트 2위는 로밍 압박의 결과물
- **Ball-Winning = 상황 카드:** 강팀 상대 Deep 블록에서 로밍 금지시킬 때만. Limited Versatility 특성 유의

## 4. 로저스 vs 틸레만스 10번 사용 구분 + 포커스

### 언제 누가 10번?
| 구성 | 사용 상황 | 성격 |
|---|---|---|
| **로저스 10번** (기본) | 오나나 결장/벤치 시 (틸레만스가 피벗으로 내려감), 뒷공간 직접 공략 원할 때 | 트랜지션·뒷공간 공략형 |
| **틸레만스 10번** | ①오나나 가용(1순위 조합) ②후반 공격 전환 카드(지고 있을 때) ③로우블록 라인 사이 창조 | 점유·라인 사이 창조형 |

- 틸레만스 본인 설명: "10번 선발 시작 vs 후반 공격 필요 시 이동 사이에서 매니저가 균형을 잡는다"
- 첼시전 사례: 틸레만스 깊은 위치 선발 → 오나나 투입되자 10번으로 이동 → 경기 반전

### 로저스 10번 포커스: Playmaker / Roaming
- **판정 근거 (3겹 검증):**
  1. 시즌 히트맵: 양 측면 + 인사이드 채널 터치, 우측 하프스페이스·박스 좌측 반복 진입
  2. 선발 분포: 좌7 / 중앙10번 7 / 투톱 2 → 좌측은 "출발점"이지 고정 아님
  3. 측면 고정 반증: 잉글랜드(투헬)에서 우측 윙 고정 시 13경기 1골로 부진
- **좌측 편중 해석:** 박스 좌측 열기 = 오른발 컷인 슈팅 도착 지점. 중앙 출발 시엔 왓킨스가 좌측으로 흐르며 로저스는 우측 포켓 점유 (상보적)
- **결론:** 기본 Playmaker/Roaming. "언테임드(untamed)" = 한 구역에 묶지 말고 자유 부여. Playmaker 정의("로밍 라이선스, 공간·기회 창출, 역습 시 노출 리스크")가 로저스 실제("최다 볼손실 255회 감수")와 정확히 일치

### 틸레만스 10번 포커스: 상황별 분기
| 상황 | 포커스 |
|---|---|
| 1순위 조합 (오나나+카마라 피벗, 틸레만스 선발 10번) | **Playmaker / Balanced** — 뒤에 2명 버티므로 10번 라인 유지가 최적 |
| 후반 공격 카드 / 3선 얇을 때 / 딥-10번 병행 | **Playmaker / Build-Up** — 내려와 빌드업 연결하는 딥 성향 재현 |

### 로저스 UT 카드별 친숙도 분기
| 모드/카드 | 최적 |
|---|---|
| 커리어 모드 | **Playmaker / Roaming** (친숙도 Playmaker+) |
| UT 기본 골드 | **Shadow Striker / Attack** (SS++, 실리 우선. 로밍은 틸레만스·마첸이 분산) |
| 고레이팅 프로모 | **Playmaker / Roaming** (전 롤 ++) |

## 5. 로저스 좌측 변형 게임플랜 (세트로 발동)

로저스를 명목상 LM에 두는 경기(부엔디아 휴식, 좌측 컷인 집중 게임플랜):
- 로저스 = **LM Inside Forward / Balanced** (LM IF+ 친숙도)
- 마첸 = **Inverted Wingback / Build-Up** (로저스가 폭 잡으니 마첸은 안쪽. 에볼루션 IWB++ 시 우선)
- ※기본 구성(로저스 중앙)에선 마첸 Wingback/Balanced 유지

## 7. FC26 롤/포커스 인게임 원문 레퍼런스 (검증에 사용한 정의)

> **주의:** 요약 사이트(fifauteam 등)에 오류 있음. 인게임 원문은 FUT.GG / FUTBIN 우선.

### 이번 검토에서 확정한 포커스 목록 (오류 정정 이력)
- **GK:** Goalkeeper(Defend/Balanced), Sweeper Keeper(Balanced/Build-Up), Ball-Playing Keeper(Build-Up)
- **풀백:** Fullback(Defend/Balanced/**Versatile**), Falseback(Defend/Balanced), Wingback(**Support/Balanced** — ※Versatile 없음), Attacking Wingback(Support/Attack), Inverted Wingback(Build-Up/Attack)
- **CB:** Defender(Defend/Balanced), Stopper(Balanced/Defend/Aggressive), **Ball-Playing Defender(Defend/Build-Up/Aggressive — 3종)**, Wide Back(Defend/Aggressive/Support)
- **CDM:** Holding(Defend/Roaming/Ball-Winning), Centre-Half(Defend), Deep-Lying Playmaker(Defend/Roaming/Build-Up), Wide Half(Build-Up/Defend), Box Crasher(Balanced)
- **CAM:** Playmaker(Balanced/Roaming/Build-Up), Shadow Striker(Attack), Half-Winger(Roaming[CAM전용]/Balanced/Attack), Classic 10(Versatile/Wide/Attack)
- **와이드 MF:** Winger(Balanced/Attack), Wide Midfielder(Defend/Support/Build-Up), Wide Playmaker(Attack), Inside Forward(Balanced/Attack)
- **ST:** Advanced Forward(Attack/Versatile/Support), Poacher(Attack/Versatile), False 9(Build-Up/Attack), Target Forward(Balanced/Attack/Wide)

### 주요 포커스 움직임 정의 (본 세팅 관련)
- **BPD/Build-Up:** 볼 소유 시 홀딩 미드필더에 가까운 위치로 전진 (파우 = 이것)
- **BPD/Aggressive:** 패스 인터셉트·압박 집중, 뒤에 공간 남김(Leave Gaps) → 파우엔 부적합(느림+하이라인 붕괴 위험)
- **BPD/Defend:** 살짝 올라가 동료 지원, Stays Back
- **Holding/Defend:** 약간 올라가 패스 옵션, Stays Back (카마라)
- **Holding/Roaming:** 피치 폭 가로지르며 패스레인 차단 + 올라가 압박 (오나나)
- **Holding/Ball-Winning:** 공격적 패스 인터셉트, Limited Versatility
- **DLP/Build-Up:** 전진해 패스 옵션 제공 (틸레만스 피벗)
- **DLP/Roaming:** 피치 폭 횡이동 지원, Out of Position 리스크
- **Playmaker/Roaming:** 자유 탐색 + 공간 창출, 역습 시 노출 (로저스)
- **Playmaker/Balanced:** 필요 시 수비 지원 + 공격 조율 (틸레만스 선발 10번)
- **Playmaker/Build-Up:** 더 깊이 내려와 받고 조직 (틸레만스 유동 10번)
- **Inside Forward/Balanced:** 주발 컷인 + 뒷공간 런/패스 옵션 번갈아 (맥긴·부엔디아)
- **Advanced Forward/Versatile:** 측면 이동해 패스 받거나 공간 창출, Build-Up Support (왓킨스)
- **Advanced Forward/Attack:** 높고 중앙 고정, Limited Width
- **Wingback/Balanced:** 볼 잃어도 전방 잔류 라이선스, 필요 시 복귀 (캐시·마첸)
- **Wingback/Support:** 안정적 소유 시에만 공격 (규율 우선)
- **Centre-Half/Defend:** 볼 소유 시 센터백 사이로 내려가 역습 차단 (린델뢰프 CDM)

## 8. 실제 25-26 전술 디테일 (게임 수동 조작 재현용)

- **빌드업 구조:** 저위 1-4-2-2-2 (GK +1 수적우위) → 고위 1-3-2-5 (중앙 우선, 와이드 1명만)
- **좌우 비대칭:** 좌측=마첸 하프스페이스 침투(로저스 컷인 시), 우측=캐시 바깥 오버랩
- **역습 타깃:** 센터백-풀백 사이 채널. 아웃렛 = 왓킨스(중앙)/로저스(와이드)
- **수비:** 4-4-2 방향성 트랩 = 한쪽으로 유도해 터치라인에 가둠. 풀백은 카운터프레스 미관여+백라인 복구
- **rest-defense:** 풀백 한 번에 한 명만 전진 (역습 실점 0의 핵심)
- **압박 트리거:** 백패스 시 1선 전진. 측면(side-on) 자세로 양방향 대비
- **세트피스:** 핵심 무기(맥피 코치). EL결승 1·2호골 모두 코너 국면. 숏코너 루틴(로저스 숏→크로스→틸레만스 쇄도)
- **스플릿 투톱 변형:** 상대가 윙어 내로잉을 풀백으로 맨투맨 추적 시 → 로저스를 왓킨스 옆까지 올려 투톱화 + 다이렉트 롱볼 (CB 와이드 스플릿 + 마르티네스 롱볼로 수동 재현)

## 10. 검증 과정에서 정정된 주요 오류 (참고 이력)

1. 맥긴/부엔디아 좌우 반전 (맥긴=우, 부엔디아=좌가 정답)
2. Wingback에 Versatile 포커스 없음 (Support/Balanced만. Versatile은 Fullback 전용)
3. 틸레만스 피벗 DLP 포커스: Roaming(횡)→Build-Up(수직) 정정
4. 왓킨스: Attack(중앙고정)→Versatile(좌우 드리프트) 정정
5. 오나나 1순위: Ball-Winning→Roaming 정정 (통계=결과지 움직임유형 아님)
6. **BPD 포커스 3종(Aggressive/Build-Up/Defend)** — 초기 "2종" 주장은 오류. 파우는 Build-Up(Aggressive는 압박형이라 부적합)
