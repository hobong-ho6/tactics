# FC 26 롤 히트맵 비교 도구 (tactics)

EA SPORTS FC 26의 **역할(Role) + 포커스(Focus)별 오프더볼 움직임 히트맵**을 fut.gg 스타일 5×5 그리드로 시각화하고, 두 전술 설정을 좌우로 나란히 비교하는 단일 파일 웹 앱입니다. 외부 의존성이 없어 `fc26-heatmap.html` 하나만 브라우저로 열면 바로 동작합니다(오프라인 가능).

## 주요 기능

- **7개 포메이션**: 4-2-3-1, 4-2-3-1 (와이드), 4-3-3, 4-4-2, 4-5-1, 4-1-2-1-2 (내로우), 3-4-3
- **전 포지션 역할/포커스**: fut.gg 기준 31개 역할·85개 포커스의 히트맵 데이터를 그대로 내장(브라우저 자동화로 스크랩한 verbatim 데이터, 우측 선수는 자동 미러링)
- **좌/우 두 패널(A·B) 독립 설정**: 포메이션·역할·포커스·표시·모드·강도를 각각 따로 조정해 비교
- **절대 스케일 합성 히트맵**: 겹치는 셀의 값을 그대로 합산(상한 없음). 단독 역할은 초록 3단계, 중첩될수록 노랑 → 주황 → 빨강
- **선수별 색 모드**: 선수마다 고유 색으로 커버리지 표시
- **셀 호버 툴팁**: 셀에 마우스를 올리면 그 칸에 겹치는 포지션·역할·기여값과 합계 표시
- **프리셋**: 현재 설정 저장(브라우저 localStorage), 어느 패널에든 불러오기, 반대편 패널로 복사
- **프리셋 내보내기/불러오기**: JSON 파일로 백업·공유(이름 중복 시 덮어쓰기 병합)

## 사용법

1. `fc26-heatmap.html`을 브라우저로 엽니다.
2. 각 패널에서 포메이션을 고르고, 하단 목록에서 선수별 역할·포커스를 지정합니다.
3. 피치 위 마커를 클릭하면 그 선수의 히트맵을 켜고 끌 수 있습니다.
4. `저장`으로 프리셋을 만들고, `불러오기`/`복사`로 A·B를 비교합니다.
5. `프리셋 내보내기`로 백업하고, 다른 환경에서 `프리셋 불러오기`로 복원합니다.

## 데이터 및 색 스케일

- 각 역할·포커스 셀 강도는 fut.gg 기준 **0.3 / 0.6 / 1.0** 세 단계입니다.
- 합성 모드는 표시되는 선수들의 셀 값을 **더해서** 색으로 표현합니다(정규화 없음, 캡 없음). 색 경계값은 `fc26-heatmap.html` 상단 스크립트의 `HEAT` 배열에서 조정할 수 있습니다.

## 참고

- 본 히트맵은 실제 경기 트래킹 데이터가 아니라 각 역할의 움직임 성향을 나타낸 참고용 모델(fut.gg 공개 데이터 기반)입니다.
